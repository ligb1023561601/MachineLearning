# MachineLearning
KNN.py includes a simple classifier of KNN which could help understande the principle of KNN algorithm
You can judge a person who has three properties if he will be in favor of a woman. 
And there is a function for handwrting digits recognition as well.
  
## Usage

1.At app startup, call the function classify_person():

``` Python
classify_person()
input the miles you travel in a year,the percentage of gaming time in one day and the ice-cream consumption in order
```

2.
- Draw a digits-image with your computer drawing tool in black and white
- Run the scripts CharImage.py to convert that image into chartext
- ‘datas.zip’ include all the test samples and training sample,please unzip it
- Run the script HandWritingNumberRecognize.py and then comes the answer
